ChatGPT + CodeMirror!

Alpha 1 Instructions: First, enter your OpenAI API key. We are not able to see this. You can get one from https://platform.openai.com/account/api-keys (Blocked on chromebooks.) Then, fill out the sentence below. If you leave the second language blank, it should use a slightly different prompt, but this is not fully tested. We reccomend using something. Press Alt + L to trigger the AI, changing your code. Use [] in a comment in either language to indicate code that you want the AI to make or change. It can be as complicated or simple as you want. ChatGPT will try to follow your directions, fix errors and translate the second language to the first. If it makes a mistake, you can press Ctrl + Z to undo and then change your code or press Alt + L again.\ Although this has many features of modern code editors, like indentation, dark mode and full undo history, it does not currently have features like saving (you have to copy it yourself), syntax highlighting (coming soon!), linting, or code running. These will all be added.
